# Linux Lab Python Homework

## Reflection

This lab involved writing a shell script to capture disk usage data and a Python
script to read and display that data. Working through the process helped reinforce
how Linux command output can be piped into files and then parsed with Python.

## Challenges

The main challenge was formatting the output from du in a way that Python could
reliably parse. Getting the column splits correct required careful attention to
whitespace handling in the script.
